Detailed Description

template<typename T>
class Polygone< T >

Classe template représentant un polygone en 2D.

La classe Polygone permet de créer et manipuler des polygones en 2D. Elle offre des méthodes pour obtenir et définir les sommets, ajouter des points, et translater le polygone.
