Detailed Description

Classe représentant une zone agricole, dérivée de ZN.

La classe ZA hérite de la classe ZN et représente une parcelle de type zone agricole. Elle permet de définir le type de la parcelle et de surcharger l'opérateur de flux pour l'affichage.
