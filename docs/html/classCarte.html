Detailed Description

Classe représentant une carte contenant des parcelles.

La classe Carte permet de charger des données de parcelles à partir d'un fichier, d'accéder à la surface totale de la carte, d'obtenir les parcelles et d'exporter les parcelles dans un fichier.
