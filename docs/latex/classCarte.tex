\doxysubsection{Detailed Description}
Classe représentant une carte contenant des parcelles. 

La classe \mbox{\hyperlink{classCarte}{Carte}} permet de charger des données de parcelles à partir d\textquotesingle{}un fichier, d\textquotesingle{}accéder à la surface totale de la carte, d\textquotesingle{}obtenir les parcelles et d\textquotesingle{}exporter les parcelles dans un fichier.
